# Python 表单自动填写脚本 🤖

一个支持 **金数据（jsj.top/jsjform）、问卷星（wjx.cn）** 等主流表单平台的自动化填写工具，实现「复制链接即自动填写」，大幅减少重复填表的繁琐操作！

## 🌟 核心功能
- **一键捕获链接**：监听系统剪贴板，复制表单链接后自动识别，无需手动输入
- **多表单适配**：兼容金数据（含 jsjform 专属结构）、问卷星等常见表单平台
- **智能字段填写**：配置文件预存填写数据，通过关键词模糊匹配自动填充（仅支持文本框）
- **友好操作提示**：实时打印填写日志，清晰标注需手动完善的字段
- **异常容错**：元素定位失败时友好提示，避免脚本卡死

## 📋 适用场景
- 批量填写重复表单（如报名问卷、数据上报、调研表格）
- 快速测试表单提交流程
- 减少人工填写误差，提升重复操作效率

## 🛠️ 技术栈
- 核心语言：Python 3.8+
- 关键依赖：
  - `selenium`：浏览器自动化（模拟点击/输入）
  - `pyperclip`：监听系统剪贴板
  - `pyyaml`：加载配置文件
  - `re`：正则表达式（识别有效链接）
- 运行环境：Chrome 浏览器 + 对应版本 ChromeDriver
