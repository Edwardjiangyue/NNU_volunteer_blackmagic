```markdown
# NNU_volunteer_blackmagic
南京师范大学抢志愿python脚本

# Python 表单自动填写脚本 🤖

一个支持 **金数据（jsj.top/jsjform）、问卷星（wjx.cn）** 等主流表单平台的自动化填写工具，实现「复制链接即自动填写文本字段」，大幅减少重复填表的繁琐操作！

## 🌟 核心功能
- **一键捕获链接**：监听系统剪贴板，复制表单链接后自动识别，无需手动输入
- **多表单适配**：兼容金数据（含 jsjform 专属结构）、问卷星等常见表单平台
- **智能文本填写**：配置文件预存填写数据，通过关键词模糊匹配自动填充文本框
- **友好操作提示**：实时打印填写日志，清晰标注需手动完善的字段
- **异常容错**：元素定位失败时友好提示，避免脚本卡死

## 📋 适用场景
- 批量填写重复表单（如报名问卷、数据上报、调研表格）
- 快速测试表单提交流程
- 减少人工填写误差，提升重复操作效率

## 🛠️ 技术栈
- 核心语言：Python 3.8+
- 关键依赖：
  - `selenium`：浏览器自动化（模拟点击/输入）
  - `pyperclip`：监听系统剪贴板
  - `pyyaml`：加载配置文件
  - `re`：正则表达式（识别有效链接）
- 运行环境：Chrome 浏览器 + 对应版本 ChromeDriver

## 🚀 快速开始

### 1. 环境准备
#### （1）安装 Python 依赖
```bash
pip install selenium pyperclip pyyaml
```

#### （2）配置 ChromeDriver
1. 查看 Chrome 浏览器版本（设置 → 关于 Chrome）
2. 下载对应版本的 ChromeDriver：[ChromeDriver 官网](https://sites.google.com/chromium.org/driver/)
3. 解压后将 `chromedriver.exe`（Windows）/ `chromedriver`（Mac/Linux）放入 Python 安装目录，或添加到系统环境变量

### 2. 配置填写数据
1. 在脚本同级目录创建 `configs.yaml` 文件
2. 按以下格式填写需要自动填充的文本信息（支持关键词模糊匹配）：
```yaml
# configs.yaml 示例
姓名: 张三
手机号: 13800138000
邮箱: example@xxx.com
公司名称: 测试公司
性别: 男
```

### 3. 运行脚本
```bash
python 抢志愿脚本.py
```
4. 复制目标表单链接（支持 jsj.top/jsjform/wjx.cn 格式），脚本将自动捕获并启动填写流程

### 核心操作流程
1. 启动脚本 → 复制表单链接
2. 脚本自动：打开浏览器 → 加载表单 → 模糊匹配文本字段并填写 → 打印填写日志
3. 手动操作：完善单选框/多选框等非文本字段 → 检查所有填写内容 → 手动点击提交按钮
4. 完成后按 Esc 键关闭浏览器，结束流程

### 字段匹配规则
- 双向模糊匹配：配置文件中的关键词（如「手机号」）与表单字段标题（如「联系电话」「手机号」）相互包含即可匹配
- 文本框处理：配置值将自动填充到对应文本输入框中（支持 text/tel 类型输入框）
- 非文本字段：单选框/多选框/下拉框/隐私条款需手动选择/勾选

## ⚠️ 注意事项
1. 仅用于个人学习、测试场景，请勿用于违规填写表单（如恶意刷量、虚假填报）
2. 部分表单可能含验证码，需手动输入验证码后继续操作
3. 若表单结构更新（如字段位置、class 变化），需调整脚本中对应的 XPath 定位表达式
4. 运行前确保 Chrome 浏览器已关闭，避免端口占用冲突
5. 脚本仅自动填写文本框，单选框/多选框需手动选择，表单需手动提交

## 🛠️ 常见问题
### Q1：脚本无法捕获链接？
- 检查是否安装 `pyperclip`：`pip install pyperclip`
- 复制链接后等待 1-2 秒，脚本会实时监听
- 确保复制的是 `https://` 开头的有效链接

### Q2：元素定位失败/填写无响应？
- 检查 ChromeDriver 版本与 Chrome 浏览器是否一致
- 确认表单链接属于支持的平台（jsj/wjx/jsjform）
- 若表单结构更新，需修改脚本中对应字段的 XPath

### Q3：配置文件不生效？
- 确保文件名为 `configs.yaml`，且编码为 UTF-8
- 配置格式需严格遵循 YAML 语法（键值对用 `: ` 分隔，不支持Tab缩进）

### Q4：浏览器版本和驱动版本不一致？
- 建议关闭浏览器自动更新，避免需要一直手动更新驱动版本
